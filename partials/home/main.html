ParallelGit is an open source project built upon JGit.

It provides your Java project with easy access to your local Git repository.


Why are we here?

No one really reads the introduction on every open source project's website. So I am going to cut it short.

Here are the things you want to know:
  • Git is not just for code. People use it to store and compare configuration files, audio samples, videos, datasets, images etc
  • There is a Java implementation of Git client called JGit
  • The JGit API is quite verbose (see examples below)
  • Things are even more difficult when you have a bare repository
ParallelGit wrapped the JGit methods to provide a more friendly API for you to use.
