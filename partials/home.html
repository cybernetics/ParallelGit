ParallelGit is an open source project built upon JGit.

It provides your Java project with easy access to your local Git repository.


Why we are here?

No one really reads the introduction on every open source project's website. So I am going to cut it short.

Here are the things you want to know:
  • Git is not just for code. People use it to store and version audio samples, videos, datasets, graphics etc
  • There is a Java implementation of Git called JGit
  • JGit is quite verbose and counter-intuitive (sometimes)
ParallelGit wrapped the JGit methods to provide a more friendly API to your project.

Examples

Create Repository
JGit ParallelGit
Create Branch (without checking out)
JGit ParallelGit
Read File
JGit ParallelGit
Copy File
JGit ParallelGit
